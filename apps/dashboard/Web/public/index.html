Install & enable notifications

This homelab is served over HTTPS with a self-signed certificate. Installing it as an app and receiving notifications requires a "secure context", so the certificate must be trusted on each device first — clicking through the browser warning is not enough.

Android (Chrome)

  1. Open the homelab HTTPS address; accept/import the certificate if prompted.
  2. Chrome menu → Install app (or Add to Home screen).
  3. Open the installed app, then allow notifications from the banner or Settings ⚙️.

iPhone / iPad (Safari, iOS 16.4+)

  1. Install the certificate profile, then enable full trust: Settings → General → About → Certificate Trust Settings.
  2. Open the homelab in Safari → Share → Add to Home Screen.
  3. Launch it from the Home Screen (push works only from the installed app), then allow notifications.

Desktop (Chrome / Edge)

  1. Trust the certificate in your OS certificate store.
  2. Use the install icon in the address bar → Install.
  3. Allow notifications from the banner or Settings ⚙️.

Don't see the notification prompt? Check its status on the Settings ⚙️ page and re-request it there.
